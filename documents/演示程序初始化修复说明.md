# Demo修复说明

## 修复的问题

### 1. Demo Launcher 和 FramelessDemoWindow 初始化顺序问题

**问题:**
```python
class DemoLauncher(FramelessWindow):
    def __init__(self):
        super().__init__()  # 调用父类初始化
        self._init_ui()     # 重复调用_init_ui()

    def _init_ui(self):
        # 这里覆盖了父类的_init_ui()
        self.setTitle("...")  # 但此时title_bar还未创建
```

**原因:**
- `FramelessWindow.__init__()` 内部调用 `self._init_ui()`
- `DemoLauncher` 覆盖了 `_init_ui()` 方法
- 父类的 `title_bar` 在 `FramelessWindow._init_ui()` 中创建
- 但子类的 `_init_ui()` 先被调用，导致 `title_bar` 不存在

**修复:**
```python
class DemoLauncher(FramelessWindow):
    def __init__(self):
        super().__init__()   # 父类_init_ui()被调用，创建title_bar
        self._demo_windows = {}
        self._setup_content()  # 不覆盖_init_ui()，使用单独的设置方法

    def _setup_content(self):  # 新方法名，不覆盖父类
        self.setTitle("PyQT Components - Demo Launcher")
        self.resize(900, 700)
        # ... 其余设置
```

## 注意事项

1. **GUI应用**: 这些demo都是GUI应用，需要图形界面环境
2. **PyQt6**: 确保已安装 PyQt6: `pip install PyQt6`

## 故障排除

如果遇到问题：

1. **ImportError**: 确保在项目根目录运行，或使用 `python -m` 方式
2. **AttributeError**: 删除 `__pycache__` 后重试
3. **显示问题**: 检查系统DPI设置和PyQt6版本兼容性
