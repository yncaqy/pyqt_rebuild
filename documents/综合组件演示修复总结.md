## 发现并修复的问题

### 1. ❌ Toast.show_anchor() 方法不存在
**位置**: 第400行
**问题**: 使用了不存在的 `show_anchor()` 方法
**修复**:
```python
# 修复前
toast.show_anchor(self)

# 修复后
toast.show(ToastPosition.TOP_RIGHT, self)
```

### 2. ❌ 缺少主题注册
**位置**: main()函数
**问题**: ThemeManager没有注册任何主题，组件无法获取样式
**修复**:
```python
# 添加导入
from themes import DARK_THEME, LIGHT_THEME, DEFAULT_THEME

# 在main()中注册
theme_mgr = ThemeManager.instance()
theme_mgr.register_theme_dict('dark', DARK_THEME)
theme_mgr.register_theme_dict('light', LIGHT_THEME)
theme_mgr.register_theme_dict('default', DEFAULT_THEME)
theme_mgr.set_theme('dark')
```

### 3. ✅ 添加应用样式
**改进**: 添加 `app.setStyle('Fusion')` 获得更好的跨平台外观

## 修复后的完整main()函数

```python
def main():
    """Run the demo application."""
    app = QApplication(sys.argv)

    # Set application style
    app.setStyle('Fusion')

    # Initialize theme manager and register themes
    theme_mgr = ThemeManager.instance()
    theme_mgr.register_theme_dict('dark', DARK_THEME)
    theme_mgr.register_theme_dict('light', LIGHT_THEME)
    theme_mgr.register_theme_dict('default', DEFAULT_THEME)
    theme_mgr.set_theme('dark')

    # Use frameless window for modern look
    window = FramelessDemoWindow()
    window.show()

    # Fallback to regular window if frameless has issues
    # window = ComponentsDemo()
    # window.show()

    sys.exit(app.exec())
```
