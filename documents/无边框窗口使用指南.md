# FramelessWindow 使用指南

## 概述

`FramelessWindow` 是一个现代化的无边框窗口组件，使用起来就像标准的 `QWidget` 一样简单！

## ✨ 核心特性

- 🎨 **主题支持** - 内置主题管理，支持运行时切换
- 🪟 **自定义标题栏** - 可自定义的标题栏和窗口控制按钮
- 🖱️ **边缘拖拽** - 支持窗口边缘调整大小
- 🚀 **高性能** - 优化的渲染和事件处理
- 🔧 **简单易用** - 与 QWidget API 兼容
- 🖥️ **跨平台** - 支持Windows、macOS、Linux

## 📦 快速开始

### 方式 1：最简单的用法（QMainWindow 风格）

```python
from src.containers.frameless_window import FramelessWindow
from PyQt6.QtWidgets import QLabel, QVBoxLayout, QWidget

# 创建窗口
window = FramelessWindow()
window.setTitle("我的应用")
window.resize(800, 600)

# 创建内容
content = QWidget()
layout = QVBoxLayout(content)
layout.addWidget(QLabel("Hello, World!"))

# 设置内容
window.setCentralWidget(content)
window.show()
```

### 方式 2：直接设置布局（QWidget 风格）✨ 推荐

```python
from src.containers.frameless_window import FramelessWindow
from PyQt6.QtWidgets import QVBoxLayout, QPushButton, QLabel

# 创建窗口
window = FramelessWindow()
window.setTitle("我的应用")

# 创建并设置布局 - 完全像 QWidget 一样！
layout = QVBoxLayout()
layout.addWidget(QLabel("直接使用 setLayout"))
layout.addWidget(QPushButton("点击我"))
window.setLayout(layout)  # 就这么简单！

window.show()
```

### 方式 3：逐个添加组件

```python
window = FramelessWindow()
window.setTitle("逐个添加")

# 使用 addWidget 方法
window.addWidget(QLabel("组件 1"))
window.addWidget(QPushButton("按钮 1"))
window.addWidget(QPushButton("按钮 2"))

window.show()
```

### 方式 4：复杂布局

```python
from PyQt6.QtWidgets import QHBoxLayout, QVBoxLayout, QPushButton

window = FramelessWindow()

# 访问内容布局
main_layout = window.contentLayout

# 添加标题
main_layout.addWidget(QLabel("复杂布局示例"))

# 创建按钮行
button_row = QHBoxLayout()
button_row.addWidget(QPushButton("确定"))
button_row.addWidget(QPushButton("取消"))

# 添加按钮行到主布局
main_layout.addLayout(button_row)

window.show()
```

## 📚 API 参考

### 核心方法

| 方法 | 说明 | 类似 QWidget |
|------|------|-------------|
| `setLayout(layout)` | 设置内容区域布局 | ✅ 完全相同 |
| `addWidget(widget)` | 添加组件到内容区域 | ✨ 便捷方法 |
| `addLayout(layout)` | 添加子布局到内容区域 | ✨ 便捷方法 |
| `setCentralWidget(widget)` | 设置中央组件 | 🔧 QMainWindow 风格 |
| `contentLayout` | 访问内容区域布局 | ✨ 属性访问 |

### 窗口控制

| 方法 | 说明 |
|------|------|
| `setTitle(title)` | 设置窗口标题 |
| `setWindowIcon(icon)` | 设置窗口图标 |
| `resize(width, height)` | 调整窗口大小 |

## 🔄 迁移指南

### 从 QWidget 迁移

```python
# 之前 - 使用 QWidget
widget = QWidget()
layout = QVBoxLayout()
layout.addWidget(QLabel("Hello"))
widget.setLayout(layout)
widget.show()

# 之后 - 使用 FramelessWindow
window = FramelessWindow()
layout = QVBoxLayout()
layout.addWidget(QLabel("Hello"))
window.setLayout(layout)  # API 完全相同！
window.show()
```

### 从 QMainWindow 迁移

```python
# 之前 - 使用 QMainWindow
main_window = QMainWindow()
main_window.setCentralWidget(content_widget)
main_window.show()

# 之后 - 使用 FramelessWindow
window = FramelessWindow()
window.setCentralWidget(content_widget)  # API 相同！
window.show()
```

## 💡 最佳实践

### 1. 使用 setLayout（推荐）

对于大多数情况，使用 `setLayout()` 最简洁：

```python
window = FramelessWindow()
layout = QVBoxLayout()
# ... 添加组件到布局
window.setLayout(layout)
```

### 2. 使用 addWidget 快速开发

快速原型开发时，直接使用 `addWidget()`：

```python
window = FramelessWindow()
window.addWidget(QLabel("标题"))
window.addWidget(QPushButton("按钮"))
```

### 3. 使用 contentLayout 构建复杂布局

需要精细控制时，访问 `contentLayout` 属性：

```python
window = FramelessWindow()
main_layout = window.contentLayout

# 添加子布局
sub_layout = QHBoxLayout()
sub_layout.addWidget(widget1)
sub_layout.addWidget(widget2)

main_layout.addLayout(sub_layout)
```

### 4. 使用 setCentralWidget 封装复杂界面

将复杂界面封装为单独的 Widget：

```python
class MyAppWidget(QWidget):
    def __init__(self):
        super().__init__()
        layout = QVBoxLayout(self)
        # ... 复杂的 UI 构建

window = FramelessWindow()
window.setCentralWidget(MyAppWidget())
window.show()
```

## 🎯 使用场景

| 场景 | 推荐方式 |
|------|---------|
| 简单窗口 | `setLayout()` |
| 快速原型 | `addWidget()` |
| 复杂布局 | `contentLayout` |
| 封装组件 | `setCentralWidget()` |

## ⚠️ 注意事项

1. **布局自动添加边距**：内容区域会自动添加 10px 边距
2. **主题继承**：子组件会自动继承主题样式
3. **窗口最小尺寸**：默认最小 400x300，可配置
4. **边缘调整**：窗口边缘 6px 区域可拖拽调整大小
