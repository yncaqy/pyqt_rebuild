# FramelessWindow 标题栏问题分析

## 问题
看不到窗口标题栏

## 原因分析

FramelessWindow 是**无边框窗口**，它使用自定义的标题栏而不是系统标题栏。

### 标题栏确实存在！
- 位置：窗口顶部
- 高度：40像素
- 颜色：深灰色 (#2d2d2d) 在dark主题下

### 为什么可能看不到？

#### 1. 颜色融合
如果内容区域的背景色和标题栏背景色相同，标题栏可能"融入"背景。

**标题栏颜色**:
- Dark主题: #2d2d2d (深灰)
- Light主题: #f0f0f0 (浅灰)

#### 2. 没有明显的分隔线
标题栏和内容之间没有边框或分隔线，看起来像一个整体。

#### 3. 窗口按钮不明显
最小化/最大化/关闭按钮：
- 使用简单符号 (─ □ ✕)
- 没有背景色，hover时才有高亮

## 标题栏特征

### 视觉特征
```
┌────────────────────────────────────────────┐
│ [图标] PyQT Components - Frameless Demo    [─][□][✕]  ← 标题栏 (40px高)
├────────────────────────────────────────────┤
│                                              │
│              内容区域                        │
│                                              │
└────────────────────────────────────────────┘
```

### 包含元素
1. **图标** (左侧): 24x24像素，默认为空
2. **标题文本**: 左对齐
3. **窗口按钮** (右侧):
   - ─ 最小化
   - □ 最大化/还原
   - ✕ 关闭

### 交互功能
- ✅ 双击标题栏 → 最大化/还原
- ✅ 拖动标题栏 → 移动窗口
- ✅ Hover按钮 → 高亮显示
- ✅ 点击 ✕ 关闭按钮 → 关闭窗口

## 如何验证标题栏存在

### 方法1: 查看窗口按钮
窗口右上角应该有三个按钮：
```
[─] [□] [✕]
```
如果能看到这些按钮，标题栏就在那里！

### 方法2: 鼠标悬停
- 将鼠标移到窗口最顶部
- 如果鼠标变成拖动图标（十字箭头）
- 或者双击顶部能使窗口最大化
- 说明标题栏存在且工作正常

### 方法3: 测试代码
创建测试脚本验证：

```python
import sys
sys.path.insert(0, 'src')

from PyQt6.QtWidgets import QApplication
from containers.frameless_window import FramelessWindow
from core.theme_manager import ThemeManager
from themes import DARK_THEME

app = QApplication(sys.argv)

# 注册主题
theme_mgr = ThemeManager.instance()
theme_mgr.register_theme_dict('dark', DARK_THEME)
theme_mgr.set_theme('dark')

# 创建窗口
window = FramelessWindow()
window.setTitle("测试窗口")
window.resize(800, 600)

# 检查标题栏
print(f"标题栏对象: {window.title_bar}")
print(f"标题栏高度: {window.title_bar.height()}")
print(f"标题栏可见: {window.title_bar.isVisible()}")
print(f"标题栏位置: {window.title_bar.pos()}")
print(f"标题栏大小: {window.title_bar.size()}")
print(f"标题文本: '{window.title_bar.title_label.text()}'")

window.show()
sys.exit(app.exec())
```

## 如果确实看不到标题栏

### 可能的原因

#### 1. 主题未正确应用
```python
# 确保在创建窗口后注册主题
theme_mgr = ThemeManager.instance()
theme_mgr.register_theme_dict('dark', DARK_THEME)
theme_mgr.set_theme('dark')

window = FramelessWindow()  # 创建窗口
```

#### 2. Content遮挡
如果setCentralWidget设置的widget覆盖了整个窗口：
```python
# FramelessWindow的布局结构：
# - content_container
#   - title_bar (40px高)
#   - content_widget (其余部分)

# setCentralWidget只设置content_widget的内容
# 不会影响title_bar
```

#### 3. 样式覆盖
如果有全局样式覆盖了标题栏样式：
```python
# 避免这样的样式：
# * { background: white; }  # 会覆盖标题栏背景

# 正确做法：
# 只为特定类设置样式
QFrame { background: white; }
```

## 让标题栏更明显

### 方案1: 添加分隔线
修改TitleBar样式，添加底边框：

```python
# 在TitleBar._build_stylesheet中添加
border-bottom: 1px solid rgba(255, 255, 255, 0.1)
```

### 方案2: 增加标题栏高度
修改配置：
```python
# WindowConfig.TITLEBAR_HEIGHT = 50  # 从40改为50
```

### 方案3: 使用更明显的颜色
修改主题：
```python
'titlebar': {
    'background': '#1a1a1a',  # 更深的颜色
    # 或
    'background': '#4a90e2',  # 使用蓝色
}
```
