## 一、核心模块与职责

一个基础播放器只需三个部分：

| 模块           | 职责                                                        |
| :------------- | :---------------------------------------------------------- |
| **视频显示区** | 渲染视频画面，通常是一个独立的控件（如 `QVideoWidget`）     |
| **控制栏**     | 提供播放/暂停、进度条、时间显示、音量、全屏、打开文件等交互 |
| **标题栏**     | （可选）显示当前文件名或窗口标题，也可与控制栏合并          |

技术选型推荐 **`QMediaPlayer` + `QVideoWidget`**，因为它是 Qt 原生组件，无需额外依赖，足以满足本地文件播放需求。

```
+------------------------------------------------------------------------+
|  [标题栏]  example_video.mp4                              [－][口][×] |  ← 可选标题栏
+------------------------------------------------------------------------+
|                                                                        |
|                               视频画面区域                             |
|                     （QVideoWidget 占满剩余空间）                      |
|                                                                        |
|                                                                        |
+------------------------------------------------------------------------+
|  ⏯  ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━  00:12 / 03:45  🔇  🔉  □  📂 |
|  播放  [       进度条滑动块          ]   当前/总时长     静音 音量 全屏 打开 |
+------------------------------------------------------------------------+
```



**布局要点**：

- 视频画面应能自适应窗口大小变化，使用 `QSizePolicy::Expanding`。
- 控制栏固定在底部，高度固定。
- 全屏时隐藏标题栏和控制栏（鼠标移动或点击时再显示）。

## 三、功能模块划分

### 1. 播放控制

- **播放/暂停按钮**：切换图标（播放▶️ / 暂停⏸️），控制 `player.play()` / `player.pause()`。
- **进度条**：水平滑动条，显示播放进度；用户拖动时跳转到指定位置。
- **时间标签**：显示“当前时间 / 总时长”，格式 `mm:ss`。

### 2. 音量控制

- **音量滑块**：水平滑动条，调节 `player.setVolume()`。
- **静音按钮**：切换 `player.setMuted()`，图标相应变化（🔊 / 🔇）。

### 3. 窗口控制

- **全屏按钮**：切换窗口全屏/正常状态，图标变化（⛶ / ❐）。
- **打开文件按钮**：弹出文件对话框，选择视频后自动播放。

## 四、关键交互逻辑（信号与槽概念）

| 触发事件             | 响应行为                         |
| :------------------- | :------------------------------- |
| 用户点击播放按钮     | 切换播放状态，更新按钮图标       |
| 媒体加载完成         | 获取总时长，设置进度条最大值     |
| 播放位置变化（每帧） | 更新进度条滑块位置、时间标签     |
| 用户拖动进度条滑块   | 设置播放位置（跳转）             |
| 音量滑块值变化       | 调整播放器音量，静音状态自动解除 |
| 静音按钮点击         | 切换静音状态，更新图标           |
| 全屏按钮点击         | 切换窗口全屏/正常，调整图标      |
| 打开文件按钮点击     | 弹出文件选择器，设置媒体并播放   |
| 播放完毕             | 停止，按钮恢复为“播放”图标       |
