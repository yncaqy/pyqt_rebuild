# FramelessWindow 易用性分析

## 当前状态：❌ 不能像 QWidget 一样开箱即用

### 主要易用性问题

#### 问题 1：需要手动设置主题
**现状：**
```python
window = FramelessWindow()  # ❌ 窗口显示空白或样式不正确
# 必须先设置主题
ThemeManager.instance().register_theme_dict("default", {...})
ThemeManager.instance().set_current_theme("default")
window = FramelessWindow()  # ✅ 才能正常显示
```

**影响：** 用户必须理解主题系统才能使用
**修复建议：** 提供默认主题或自动加载

---

#### 问题 2：需要手动调用 setCentralWidget()
**现状：**
```python
window = FramelessWindow()  # ❌ 空窗口，没有内容
window.show()  # 显示空白窗口

# 必须手动设置内容
widget = QLabel("Hello")
window.setCentralWidget(widget)  # ✅ 才有内容
```

**影响：** 无法像 QWidget 一样直接添加子组件
**修复建议：**
- 支持 `setLayout()` 直接设置布局
- 或者在 `__init__` 中接受内容 widget 参数

---

#### 问题 3：没有默认尺寸
**现状：**
```python
window = FramelessWindow()
window.show()  # ❌ 窗口极小或不显示
# 必须手动设置尺寸
window.resize(400, 300)  # ✅ 设置默认尺寸
```

**影响：** 不设置尺寸窗口不可见
**修复建议：** 提供合理的默认尺寸（如 800x600）

---

#### 问题 4：初始化复杂
**现状：**
```python
window = FramelessWindow()  # ❌ 需要多步设置

# 需要额外设置
window.setTitle("My Window")
window.resize(400, 300)
window.setCentralWidget(content)
window.show()
```

**影响：** 使用体验不如 QWidget
**修复建议：** 在 `__init__` 中支持更多参数

---

#### 问题 5：事件过滤器可能干扰
**现状：**
```python
# FramelessWindow 在所有子组件上安装事件过滤器
# 可能影响用户自定义组件的事件处理
```

**影响：** 用户需要理解事件过滤器机制
**修复建议：**
- 提供选项禁用事件过滤器
- 或提供白名单机制

---

#### 问题 6：没有内容占位符
**现状：**
```python
window = FramelessWindow()
window.show()  # ❌ 内容区域完全空白
```

**影响：** 看起来像 bug
**修复建议：** 显示占位符或提示文本

---

#### 问题 7：中心内容需要手动设置边距
**现状：**
```python
content = QLabel("Text")
window.setCentralWidget(content)  # ❌ 文本贴边显示
# setCentralWidget 内部有固定的 10px 边距，但用户不知道
```

**影响：** 用户可能手动重复设置边距
**修复建议：** 文档化或提供参数控制

---

## 建议的改进方案

### 改进 1：支持开箱即用的默认主题

```python
class FramelessWindow(QWidget):
    def __init__(self, parent=None):
        super().__init__(parent)

        # 自动注册默认主题
        theme_mgr = ThemeManager.instance()
        if not theme_mgr.current_theme():
            from themes import DEFAULT_THEME
            theme_mgr.register_theme_dict("default", DEFAULT_THEME)
            theme_mgr.set_current_theme("default")
```

**优先级：** 🔴 高 - 阻碍基本使用

---

### 改进 2：支持构造函数参数

```python
class FramelessWindow(QWidget):
    def __init__(
        self,
        parent=None,
        title: str = "",           # 新增：窗口标题
        size: tuple = (800, 600),  # 新增：默认尺寸
        content_widget=None,         # 新增：内容组件
        central_widget=None          # 新增：中心组件（别名）
    ):
        super().__init__(parent)

        # 设置标题
        if title:
            self.setTitle(title)

        # 设置尺寸
        if size:
            self.resize(*size)

        # 设置内容
        widget = content_widget or central_widget
        if widget:
            self.setCentralWidget(widget)
```

**使用示例：**
```python
# 方式 1：简单使用
window = FramelessWindow(
    title="My Window",
    size=(600, 400),
    content_widget=QLabel("Hello World")
)
window.show()

# 方式 2：完全开箱
window = FramelessWindow()  # 使用默认值
window.show()
```

**优先级：** 🟡 中 - 显著改善使用体验

---

### 改进 3：支持直接设置布局

```python
class FramelessWindow(QWidget):
    def setLayout(self, layout):
        """Allow direct layout setting on content widget."""
        self.content_widget.setLayout(layout)
        return super().setLayout(layout)
```

**使用示例：**
```python
window = FramelessWindow()

# 方式 1：使用 setCentralWidget（现有）
window.setCentralWidget(widget)

# 方式 2：直接使用 setLayout（新增）
layout = QVBoxLayout()
layout.addWidget(QLabel("Hello"))
window.setLayout(layout)  # ← 更符合 Qt 习惯
```

**优先级：** 🟡 中 - 更符合 Qt 使用习惯

---

### 改进 4：添加占位符内容

```python
class FramelessWindow(QWidget):
    def _init_ui(self):
        ...

        # 如果没有内容，显示占位符
        if not self._content_placeholder_shown:
            self._show_content_placeholder()

    def setCentralWidget(self, widget):
        # 移除占位符
        if hasattr(self, '_placeholder_label'):
            self._placeholder_label.deleteLater()

        # 添加实际内容
        layout = self.content_widget.layout()
        if layout:
            while layout.count():
                item = layout.takeAt(0)
                widget = item.widget()
                if widget:
                    widget.deleteLater()
        new_layout = QVBoxLayout(self.content_widget)
        new_layout.setContentsMargins(10, 10, 10, 10)
        new_layout.addWidget(widget)
```

**优先级：** 🟢 低 - 视觉改进，不影响功能

### 改进 5：改进事件过滤器安装

```python
class FramelessWindow(QWidget):
    def __init__(
        self,
        parent=None,
        install_event_filters=True  # 新增：可选择是否安装事件过滤器
    ):
        super().__init__(parent)
        self._install_event_filters_flag = install_event_filters
        ...

    def _install_event_filters(self):
        """Install event filters if enabled."""
        if not self._install_event_filters_flag:
            return

        # 现有逻辑
        widgets = [...]
        for widget in widgets:
            if widget:
                widget.installEventFilter(self)
```

**优先级：** 🟢 低 - 高级功能，可选

---

### 改进 6：提供便捷方法

```python
class FramelessWindow(QWidget):
    def addWidget(self, widget):
        """Convenience method to add widget to content area."""
        layout = self.content_widget.layout()
        if not layout:
            layout = QVBoxLayout(self.content_widget)
        layout.addWidget(widget)

    def addLayout(self, layout):
        """Convenience method to add layout to content area."""
        current = self.content_widget.layout()
        if current:
            # 合并布局
            current.addLayout(layout)
        else:
            self.content_widget.setLayout(layout)
```

**优先级：** 🟢 低 - 便捷方法，非必需

## 优先级排序

| 优先级 | 改进 | 影响 | 实现难度 |
|--------|------|------|---------|
| 🔴 高 | 默认主题 | 阻碍基本使用 | 简单 |
| 🔴 高 | 构造函数参数 | 显著改善体验 | 简单 |
| 🟡 中 | 默认尺寸 | 改善可见性 | 简单 |
| 🟡 中 | 直接设置布局 | 更符合习惯 | 简单 |
| 🟢 低 | 占位符内容 | 视觉改进 | 中等 |
| 🟢 低 | 事件过滤器选项 | 高级功能 | 简单 |
| 🟢 低 | 便捷方法 | 提升体验 | 简单 |

## 总结

### 当前状态：❌ 不能开箱即用

**必需改进（P0）：**
1. ✅ 提供默认主题或自动注册
2. ✅ 支持构造函数参数（title, size, content）

**推荐改进（P1）：**
3. ✅ 提供合理的默认尺寸
4. ✅ 支持直接 setLayout()

**可选改进（P2）：**
5. ⚪ 添加占位符内容
6. ⚪ 事件过滤器选项
7. ⚪ 便捷方法（addWidget）

### 改进后目标

    title="My Window",
