# 主题切换崩溃问题修复

## 问题描述

点击demo中的主题切换按钮会导致程序崩溃。

## 根本原因

**方法名不匹配:**

- **ThemeManager** 中的方法名是 `set_current_theme(theme_name)`
- **所有Demo** 中使用的方法名是 `set_theme(theme_name)`

当demo调用 `ThemeManager.instance().set_theme('dark')` 时，AttributeError被抛出，导致程序崩溃。

## 解决方案

在 `ThemeManager` 类中添加 `set_theme()` 作为 `set_current_theme()` 的便捷别名方法：

```python
def set_theme(self, theme_name: str) -> None:
    """
    Set active theme (convenience method).

    This is a convenience alias for set_current_theme().
    Use this method for a simpler API.

    Args:
        theme_name: Name of theme to activate

    Raises:
        ValueError: If theme not registered

    Example:
        ThemeManager.instance().set_theme('dark')
    """
    self.set_current_theme(theme_name)
```
